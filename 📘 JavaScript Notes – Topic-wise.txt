1Ô∏è‚É£ Variables (var, let, const)
var

Function-scoped

Can be re-declared & updated

Gets hoisted (initialized with undefined)

var a = 10;
var a = 20; // allowed

let

Block-scoped

Cannot be re-declared in the same scope

Can be updated

Hoisted but not initialized (Temporal Dead Zone)

let x = 10;
x = 20; // allowed

const

Block-scoped

Cannot be re-declared or updated

Must be initialized at declaration

const pi = 3.14;

3Ô∏è‚É£ Functions & Arrow Functions
Normal Function

  return a + b;
}

Function Expression

  return a + b;
};

Arrow Function

Differences

Arrow functions do NOT have their own this

Cannot be used as constructors

4Ô∏è‚É£ Scope (Block, Function, Global)
Global Scope

Accessible everywhere.

Function Scope

Variables declared with var inside function ‚Üí only inside that function.

function test() {
  var x = 10; // function-scoped
}

Block Scope

Variables declared with let and const inside {}.

{
  let a = 5; // block-scoped
}

5Ô∏è‚É£ Hoisting

Hoisting = variable & function declarations are moved to the top.

Example:
console.log(a); // undefined
var a = 5;

let/const hoisting

They are hoisted but not initialized ‚Üí ReferenceError.
